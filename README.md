# Space Tourism Website

A responsive multi-page space tourism website built with React and Vite.

## Description

This project is an interactive space tourism website. It features multiple pages that allow users to explore different space destinations, meet the crew, and learn about the technologies used in space travel.

## Technologies Used

- **React** - JavaScript library for building user interfaces
- **Vite** - Next-generation frontend tooling
- **CSS/HTML** - Styling and markup
- **Google Fonts** - Uses Bellefair, Barlow Condensed, and Barlow fonts

## Setup and Installation

To run this project locally, follow these steps:

1. Clone the repository to your local machine.
2. Open the project directory in your terminal.
3. Run `npm install` to install the required dependencies.
4. Run `npm run dev` to start the local development server.
5. Open the provided `localhost` link in your browser to view the site.
